About This Website

Why Transformers?

Growing up, Transformers was one of my favorite cartoons and movie franchises. What made it so exciting was seeing ordinary vehicles transform into powerful robots, each with their own personality, strengths, and purpose.

The rivalry between the Autobots and Decepticons created unforgettable battles while teaching lessons about leadership, teamwork, courage, and perseverance. Characters like Optimus Prime inspired viewers to always do what is right, while villains like Megatron showed how ambition without honor can lead to destruction.

About This Project

This website was created as a static web application using HTML, CSS, and JavaScript. It features several Transformer character profiles, navigation between multiple pages, and interactive features such as a search bar and character submission form.
